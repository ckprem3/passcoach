What is a password?

In a narrower IT concept a password can be defined as an array characters intended to differentiate an authorized entity from an unauthorized entity.

It is used for granting access to different resources for the holder entities.


Types of passwords

1. Text Password: A string of characters used to authenticate a user, such as alphanumeric passwords, passphrases, and PINs. This represents the above definition and will be the focus of this page

2. Biometric Password: Authentication based on a user's unique physical characteristics, such as fingerprints, facial recognition, or voice recognition.

3. Token-based Authentication: Authentication based on a unique physical token such as a smart card, hardware token or key fob.

4. Multi-Factor Authentication (MFA): Authentication that requires more than one method of authentication, such as a password and a fingerprint or facial recognition scan.

5. One-Time Password (OTP): A password that is valid for only one login session or transaction and can be generated either by software or hardware.

6. Pattern Password: Authentication based on a specific pattern or shape, such as drawing a pattern on a touch screen device.

7. Keystroke Dynamics: Authentication based on how the user types their password, including time between keystrokes and typing speed.

8. Smart Card Authentication: Authentication that requires an electronic card or USB device containing a digital identity certificate.
